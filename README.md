# Daniel Ore – Portfolio

My personal portfolio website where I showcase my projects and skills as a Frontend Developer.  
Built with HTML, TailwindCSS, and JavaScript.

## About

I built this portfolio to give potential employers a quick overview of who I am, what I've worked on, and how to get in touch. It's a single-page site with sections for projects, skills, and contact info.

## Projects featured

- **Gardeco POS System** – Frontend for a point-of-sale system with external API integration
- **SnabbMat** – Recipe app with hundreds of quick and easy recipes
- **ChatFlow** – Full-stack chat app with authentication and an AI-powered chatbot
- **DishApp** – Recipe browsing app with favorites saved in local storage

## Technologies used

- HTML5
- TailwindCSS (via CDN)
- JavaScript ES6+
- Font Awesome icons

## Running locally

Just open `index.html` in a browser — no build step needed.
